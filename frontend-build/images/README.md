# Student Images for Homepage

## How to Add Your Runda TSS Student Images

1. **Download images from Runda TSS website** (http://rundatss.rw/)
2. **Save images in this folder** with these names:
   - `student1.jpg`
   - `student2.jpg`
   - `student3.jpg`
   - `student4.jpg`
   - `student5.jpg`
   - `student6.jpg`

3. **Image Requirements:**
   - Format: JPG, PNG, or WebP
   - Recommended size: 1200x800px or similar aspect ratio
   - File size: Under 500KB each for fast loading

4. **Placeholder Images:**
   - The system will show gradient placeholders until you add real images
   - Replace placeholders with actual student photos for best effect
