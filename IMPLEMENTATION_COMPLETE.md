   - Synonym detection

   - Context analysis
3. Provides score + confidence (0.0-1.0)

- **0.90-1.00**: Very confident - 95%+ accurate

- **0.60-0.74**: Moderate - 85%+ accurate
- **0.40-0.59**: Low - needs review
- **0.00-0.39**: Very low - definitely review
